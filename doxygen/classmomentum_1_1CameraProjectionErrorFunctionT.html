Detailed Description

template<typename T>
class momentum::CameraProjectionErrorFunctionT< T >

Error function that projects 3D skeleton points through a bone-parented camera using an IntrinsicsModelT<T> and penalizes reprojection error in pixel space.

The camera is rigidly attached to a skeleton joint (cameraParent) with a fixed offset (cameraOffset). If cameraParent is kInvalidIndex, the camera is static in world space (cameraOffset is used directly as eyeFromWorld).

Each constraint specifies a 3D point attached to a joint and a target 2D pixel. The error is the squared distance between the projected point and the target.
