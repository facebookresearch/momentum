Detailed Description

template<typename T>
struct momentum::SupportPlaneT< T >

Oriented plane used to classify support contacts and project support polygons.

The default is Momentum's Y-up ground plane. Its in-plane basis intentionally preserves the existing world-XZ support-polygon coordinates: uAxis is +X and vAxis is +Z. That makes the default basis left-handed, so callers should treat (u, v) as projection coordinates, not as a right-handed frame. Points with signed distance <= the caller's margin are on the contact side of the plane.
