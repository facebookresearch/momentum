Detailed Description

template<typename T>
class momentum::JerkSequenceErrorFunctionT< T >

Error function that penalizes the jerk (third derivative) of joint positions across four consecutive frames using a standard finite difference stencil [1, -3, 3, -1].

The jerk residual for each joint is computed as: jerk = pos[t-1] - 3*pos[t] + 3*pos[t+1] - pos[t+2] - targetJerk

This is useful for enforcing smoothness in acceleration changes, which produces more natural-looking motion. By default, the target jerk is zero, which penalizes any non-zero jerk (smoothness constraint on acceleration).

Note: This error function only constrains position jerk, not rotation jerk, as rotation derivatives involve significantly more complex mathematics.
