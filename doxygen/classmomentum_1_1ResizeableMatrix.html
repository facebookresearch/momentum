Detailed Description

template<typename T>
class momentum::ResizeableMatrix< T >

A matrix class that can be resized without reallocating memory when shrinking.

The native Eigen matrix type reallocates the matrix every time you call either resize() or conservativeResize(), which means there is no way to have a matrix that reuses memory. This is inconvenient, because there are definitely cases where we want to be able to reuse the same memory over and over for slightly different matrix sizes (for example, if we are computing our Jacobian blockwise).
