Detailed Description

template<typename S>
struct momentum::SDFColliderT< S >

SDF collision geometry for character collision detection.

Defined by a transformation, parent joint, and a shared pointer to a SignedDistanceField. The SDF data is shared to avoid expensive copies of large SDF volumes.
