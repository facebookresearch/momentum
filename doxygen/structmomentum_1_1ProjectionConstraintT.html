Detailed Description

template<typename T>
struct momentum::ProjectionConstraintT< T >

Per-constraint data for CameraProjectionErrorFunctionT.

Each constraint specifies a 3D point (attached to a skeleton joint via parent + offset) and a target 2D pixel location. The error function penalizes the reprojection error between the projected 3D point and the target pixel.
