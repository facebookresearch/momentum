Detailed Description

template<typename T>
struct momentum::JointToJointPositionDataT< T >

Constraint data for joint-to-joint position error between two joints.

This constraint measures the position of a point attached to the source joint in the coordinate frame of the reference joint, and compares it to a target position (also expressed in the reference joint's coordinate frame).
