Detailed Description

template<typename T>
class momentum::PointTriangleVertexErrorFunctionT< T >

Support constraining different parts of the mesh together by specifying that a source vertex should target a location on the mesh defined by a target triangle and its normal.

The target point is specified as sum_i (bary_i * target_triangle_vertex_i) + depth * target_triangle_normal. Note that this constraint applies "forces" both to the source vertex and every vertex in the target triangle, so it will actually try to pull both toward each other.
