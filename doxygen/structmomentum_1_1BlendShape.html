Detailed Description

Skinning class that combines blend shape vectors with a base shape.

Extends BlendShapeBase by adding a base shape (e.g., PCA mean) and functionality for computing deformed shapes and estimating blend coefficients. Primarily used to model identity-dependent shape deformations.
