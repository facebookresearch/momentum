Detailed Description

template<typename T>
class momentum::SkinnedLocatorTriangleErrorFunctionT< T >

Support constraining a skinned locator to a location on the mesh defined by a target triangle and its normal.

The target point is specified as sum_i (bary_i * target_triangle_vertex_i) + depth * target_triangle_normal. This constraint applies "forces" both to the skinned locator and every vertex in the mesh target triangle, so it will actually try to pull both toward each other.
