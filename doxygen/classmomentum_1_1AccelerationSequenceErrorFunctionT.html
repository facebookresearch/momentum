Detailed Description

template<typename T>
class momentum::AccelerationSequenceErrorFunctionT< T >

Error function that penalizes the acceleration of joint positions across three consecutive frames using a standard finite difference stencil [-1, 2, -1].

The acceleration residual for each joint is computed as: accel = pos[t+1] - 2*pos[t] + pos[t-1] - targetAcceleration

This is useful for ballistic motion constraints where joints should follow a specific acceleration (e.g., gravity). By default, the target acceleration is zero, which penalizes any non-zero acceleration (smoothness constraint on velocity).

Note: This error function only constrains position acceleration, not rotation acceleration, as rotation acceleration involves significantly more complex mathematics.
