# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-XX-XX

First stable release of the Momentum library.

### Highlights

- Forward/inverse kinematics with interpretable parameterization
- Body tracking solvers (RGBD, monocular RGB)
- Motion sequence optimization and marker tracking
- File format support: FBX, GLTF, C3D, USD
- Python bindings with PyTorch integration
- CPU and GPU (CUDA) acceleration
- Cross-platform: Linux, macOS, Windows

### Distribution

- PyPI: `pymomentum-cpu`, `pymomentum-gpu`
- conda-forge: `pymomentum`, `momentum-cpp`
- Python 3.12, 3.13 / PyTorch 2.8+
