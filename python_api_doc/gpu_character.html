class pymomentum.gpu_character.AdvancedInverseParameterTransform(parameter_transform: ParameterTransform, rcond: float = 1e-05)

Bases: InverseParameterTransform

This is a more advanced version of InverseParameterTransform that can handle the case where pose and scale are mutually exclusive (i.e. no joint parameter is affected by both pose and scale). In this case, we can compute a pose-only SVD and use that to compute the inverse parameter transform.

skin_with_transforms(translation: Tensor, rotation: Tensor, scale: Tensor, rest_vertex_positions: Tensor) → Tensor

Apply skinning using matrices rather than skeleton states. This can be useful in cases where you directly predict the matrices but want to avoid extra conversions.

Parameters:

  • translation – A tensor of shape (…, 3) representing the translation.

  • rotation – A tensor of shape (…, 3, 3) representing the 3x3 rotation matrices.

  • scale – A tensor of shape (…, 1) representing the scale.

  • rest_vertex_positions – A tensor of shape (…, 3) representing the rest vertex positions.

Returns:

A tensor of shape (…, 3) representing the skinned vertex positions.
